# 电子商务用户行为分析

## 一、项目背景

本项目基于公开电商数据集，围绕用户购买行为、品牌销售、商品类别、价格区间和时间趋势进行分析，使用 Excel 完成数据清洗、指标构建、数据透视分析和可视化看板制作，并结合业务场景提出运营优化建议。

## 二、业务问题

本项目主要围绕以下问题展开：

1. 哪些品牌销售额最高，值得重点运营？
2. 哪些商品类别贡献了主要销售额？
3. 用户是直接购买多，还是加购后未转化的行为多？
4. 哪些时间段用户购买行为更活跃？
5. 哪些商品销售额最高，适合重点推荐？
6. 不同价格区间商品的销量和销售额有什么差异？

## 三、数据来源

数据来源于公开电商数据集，包含用户 ID、商品 ID、品牌、商品名称、商品类别、商品类目 ID、行为类型、时间戳、售价、销量等字段。

## 四、数据处理

使用 Excel 对原始数据进行处理，主要包括：

1. 处理 CSV 中文编码问题；
2. 新增销售额字段：销售额 = 售价 × 销量；
3. 从时间戳中提取日期和小时字段；
4. 按售价划分价格区间；
5. 将 buy、addbuy 行为用于购买与加购行为分析；
6. 通过数据透视表构建品牌、品类、时间、价格区间等分析维度。

## 七、核心业务结论

1. 根据品牌销售额分析，Samsung、HP、Microsoft 等品牌销售额排名靠前，说明平台销售额主要集中在头部品牌。
2. 根据商品类别销售额占比，笔记本电脑和智能手机合计贡献超过一半销售额，是平台核心销售品类。
3. 根据用户行为类型占比，购买行为占 84%，加购行为占 16%，整体转化表现较好。
4. 根据月度销售额趋势，11 月销售额最高，结合现实电商场景，可能受到“双11”“黑色星期五”等促销节点影响。
5. 根据价格区间销量与销售额分析，中价商品销量最高，高价商品销售额贡献较大，应分别采取规模化销售和转化提升策略。

## 八、运营优化建议

1. 对 Samsung、HP、Microsoft 等高销售额品牌增加推荐位和活动曝光。
2. 针对笔记本电脑、智能手机等核心品类设置专题活动和组合套餐。
3. 对加购用户推送优惠券、限时折扣和购物车提醒，提高购买转化。
4. 结合用户活跃时间段安排促销推送，提高营销触达效率。
5. 对高价商品采用分期、满减、赠品组合等方式降低购买门槛。
